## 一、基础操作

### 1.1 行内公式

符号：`$`公式`$`，与文本连接时打上半角空格。

| 有半角空格 | 无半角空格 |
|-------|-------|
| 有 $T$ 组数据。 | 有$T$组数据。 |
| two integers$n$and$m$ | two integers $n$ and $m$ |

### 1.2 独行公式

符号：`$$`公式`$$`，公式会居中显示在单独的一行。

### 1.3 上标下标

上标 `^`，下标 `_`。

| 代码 | 显示效果 |
|-------|-------|
| a^b| $a^b$ |
| w_1 | $w_1$ |
| A_5^3 | $A_5^3$ |

### 1.4 组合

上标、下标和其他操作只适用于下一个“组”，“组”可以是单个符号，当组内符号不止一个时，需要组合符号，语法：`{`公式`}`。

| 代码 | 显示效果 |
|-------|-------|
| 10^10 | $10^10$ |
| 10^{10} | $10^{10}$ |
| x_i^2 | $x_i^2$ |
| x_{i^2} | $x_{i^2}$ |

同时，大括号还能消除二义性，如：`$x^5^6$` 将得到一个错误，必须使用大括号来界定 ^ 的结合性。

| 代码 | 显示效果 |
|-------|-------|
| `{x^5}^6` | ${x^5}^6$ |
| `{x^5}^6` | $x^{5^6}$ |

## 二、字体

### 2.1 文本字体

Markdown文件中文字的书写与一般的文字编辑软件一致，直接编写内容即可，默认为正体，可以在此基础上进行简单设置。

| 代码 | 显示效果 |
| ------ | ------ |
| <code>*斜体*</code> | *斜体* |
| <code>**粗体**</code> | **粗体** |
| <code>***粗斜体***</code> | ***粗斜体*** |
| <code>~~删除~~</code> | ~~删除~~ |

### 2.2 公式字体

公式内的字母默认为数学斜体，在使用微分符号 $\mathrm d$、二项式系数$\mathrm C$、等于号上的 $\mathrm {def}$、自然常数 $\mathrm e$、虚数单位 $\mathrm i$ 等情况下，需要把这些字母改为正体。

语法为 \mathrm{符号}

| 斜体 | 正体 |
|-------|-------|
| $dx$ | $\mathrm {d} x$ |
| $C_n^m$ | $\mathrm {C}_n^m$ |
| $\stackrel{def}{=}$ | $\stackrel{\mathrm{def}}{=}$ |
| $e$ | $\mathrm e$ |
| $i$ | $\mathrm i$ |

其他字体详见[附录-字体](#1字体)

## 三、占位符、定界符、格式控制

### 3.1 占位符

| 符号 | 语法 | 代码 |显示效果 |
| ----- | ----- | ------ | ----- |
| 无占位符 | 无 | A B | $A B$ |
| 空格 | \ | A \ B | $A \ B$ |
| quad空格 | \quad| A \quad B | $A \quad B$ |
| 2个quad空格 | \qquad | A \qquad B | $A \qquad B$ |

### 3.2 定界符

#### 3.2.1 括号

| 代码 | 显示效果 |
| ------ | ----- |
| f(x) | $f(x)$ |
| f(g(x)) | $f(g(x))$ |
| 1) | $1)$ |

可以使用不同大小的括号来面对各种场景。

代码：`() \big( \big) \Big(\Big) \bigg(\bigg) \Bigg(\Bigg)`

$ () \big( \big) \Big( \Big) \bigg( \bigg) \Bigg( \Bigg)$

例如：`\Bigg( \bigg( \Big( \big( (x) \big) \Big) \bigg) \Bigg)`，显示效果为：

$\Bigg( \bigg( \Big( \big( (x) \big) \Big) \bigg) \Bigg)$

#### 3.2.2 自适应大小括号

也可以使用自适应括号，它们会自动变成合适大小的括号：\left( \right)

| 代码 |显示效果 |
| ------ | ----- |
| \left( x \right) | $\left (x \right)$ |
| \left( \Big( x \Big) \right) | $\left( \Big( x \Big) \right) $ |
| \left( \Bigg( x \Bigg) \right) | $\left (\Bigg(x \Bigg) \right)$|

若自适应括号只有一半，另一半不能忽略，需要用一个`.`替代。

| 代码 |显示效果 |
| ------ | ----- |
| x \right) | 无法解析 |
| \left. x \right) | $\left. x \right)$ |
| \left( x^2 \right. | $\left( x^2 \right.$|

#### 3.2.3 中括号、大括号

中括号与大括号的语法类似，不过大括号会和组合符号冲突，需要转义。

| 代码 |显示效果 |
| ------ | ----- |
| [x] | $[x]$ |
| \\{ x \\}| $$ \{ x \} $$ |
| \left\\{ \Big[ x \Big] \right \\} | $$\left \{ \Big[ x \Big] \right \} $$|

| \underbrace {a+\overbrace {b+\dots+b}^{m个}}_{n个} | $\underbrace {a+\overbrace {b+\dots+b}^{m个}}_{n个}$ |

### 3.4 堆积符号

| 语法 | 代码 | 显示效果 | 备注 |
| ----- | ----- | ------ | ----- |
| \stacrel{上位符号}{基位符号} | \vec{x}\stackrel{\mathrm{def}}{=}{x_1,\dots,x_n} | $\vec{x}\stackrel{\mathrm{def}}{=}{x_1,\dots,x_n}$ | 基位符号大，上位符号小|
| {上位公式\atop 下位公式} | \sum_{k_0,k_1,\ldots>0 \atop k_0+k_1+\cdots=n}A_{k_0}A_{k_1}\cdots | $\sum_{k_0,k_1,\ldots>0 \atop k_0+k_1+\cdots=n}A_{k_0}A_{k_1}\cdots$ | 上下符号一样大，两侧大括号不能省略 |
| {上位公式\choose 下位公式} | \overbrace{1+1+...+1}^{n个} = n | ${n+1 \choose k}={n \choose k}+{n \choose k-1}$ | 同上，且上下符号被包括在圆弧内 |

$\vec{x}\stackrel{\mathrm{def}}{=}{x_1,\dots,x_n}\\ {n+1 \choose k}={n \choose k}+{n \choose k-1}\\ \sum_{k_0,k_1,\ldots>0 \atop k_0+k_1+\cdots=n}A_{k_0}A_{k_1}\cdots$

#### 4.2.1 分式嵌套

分式可以嵌套，使用`\frac`和`\cfrac`会得到两种不同效果，如：

$$x = a_0 + \frac{1^2}{a_1 + \frac{2^2}{a_2 + \frac{3^2}{a_3 + \frac{4^4}{a_4 + \cdots}}}}$$

`$$x = a_0 + \cfrac{1^2}{a_1 + cfrac{2^2}{a_2 + \cfrac{3^2}{a_3 + \cfrac{4^4}{a_4 + \cdots}}}}$$`

$$x = a_0 + \cfrac{1^2}{a_1 + \cfrac{2^2}{a_2 + \cfrac{3^2}{a_3 + \cfrac{4^4}{a_4 + \cdots}}}}$$

<font face="楷体" color=green size=4>数学符号、单位符号、常量、数字码或上下标注释的英文字母使用正体。</font>

$\mathrm {ABCDEFGHIJKLMNOPQRSTUVWXYZ}$
